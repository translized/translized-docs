# Get Key

Adds key to project.

Request type: <span style={{ color: '#167219' }}>_POST_</span>

### API Endpoint

```url
https://api.translized.com/term/get
```

### Parameters

| Param        | Type   | Required | Description                                                                                                                      |
| ------------ | ------ | -------- | -------------------------------------------------------------------------------------------------------------------------------- |
| projectId    | String | True     | Id of your project                                                                                                               |
| termKey      | String | True     | Key of key to add                                                                                                                |
| languageCode | String | False    | Language code. If defined translation will be returned only for that language. If not defined all key translations are returned. |
