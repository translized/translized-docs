# Available languages

Returns list of all available languages.

:::info Note
List of all supported languages with their respective codes can be found in our [Supported Languages](/docs/features/supported-languages) section.
:::

Request type: <span style={{ color: '#167219' }}>_POST_</span>

### API Endpoint

```url
https://api.translized.com/language/available
```

### Parameters

None

### Example requests

<Tabs className="api-tabs">
<TabItem value="cURL" label="cURL">

```curl
curl -X POST -H "Content-Type: application/json" \
     -H "api-token: 42cf5fec-a74c-4a53-8ebc-86a4e52b7ce6" \
     https://api.translized.com/language/available
```

</TabItem>
<TabItem value="js-fetch" label="Javascript Fetch">

```js
fetch('https://api.translized.com/language/available', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'api-token': '42cf5fec-a74c-4a53-8ebc-86a4e52b7ce6',
  },
})
  .then((response) => response.json())
  .then((data) => console.log(data))
  .catch((error) => console.error(error));
```

</TabItem>
<TabItem value="js-axios" label="Javascript Axios">

```js
const response = await axios.post(
```
