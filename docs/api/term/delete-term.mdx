# Delete Term

Deletes Term.

Request type: <span style={{ color: '#167219' }}>_POST_</span>

### API Endpoint

```url
https://api.translized.com/term/delete
```

### Parameters

| Param     | Type   | Required | Description           |
| --------- | ------ | -------- | --------------------- |
| projectId | String | True     | Id of your project    |
| termKey   | String | True     | Key of term to delete |
