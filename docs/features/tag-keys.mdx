Key Tags allow you to easily tag (mark) keys, based on your preferences (”to do”, “review” etc.), and simplify your team’s collaboration.

## Add Key Tags

<aside>
💡 Only the Owner and Admin can create new Key Tags. Contributors don’t have access to Tags.

</aside>

To create a new Key Tag, go to Settings on the left hand side, and under ‘Key Tags’, click ‘Add’.

Name the tag, add the color and you’re all set.

![Screen Shot 2022-08-19 at 6.46 1.png](/features/tag-keys/Screen_Shot_2022-08-19_at_6.46_1.png)

## Tag Keys

Once your tags have been created, you can easily tag your keys, by editing the key. You can add one or more tags to each key.

![Screen Shot 2022-08-19 at 6.49 1.png](/features/tag-keys/Screen_Shot_2022-08-19_at_6.49_1.png)

## Filter By Tags

To easily navigate your keys, you can filter them by tags.
