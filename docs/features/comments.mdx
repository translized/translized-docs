Translized makes it easy for you to communicate with your team members through Comments and Mentions.

## Add Comments

To add a comment, go to key and on the right hand side, you’ll find the comments section.

## Mention Users

To mention a User, start with “@” and the proceed by adding the name of the team member to notify them.

To mention "everyone" in the team, type start with "#" and then select "everyone" from the dropdown.
