### Nested

![json-example-nested.png](/features/file-formats/json-example-nested.png)

Keys presented in the nested json example will be represented inside Translized as keys:

NAVBAR::HOW_TO_USE - How to use

NAVBAR::LOGIN - Login

NAVBAR::SIGN_UP - Signup

LANDING::TEXT_PART_1 - Lorem ipsum dolor sit amet, consectetur adipiscing elit

LANDING::TEXT_PART_2 - Lorem ipsum dolor sit amet, consectetur adipiscing elit

"Double two-dots" (::) will be used as a separator for the nested keys inside your json file.
