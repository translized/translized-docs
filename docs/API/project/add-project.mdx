# Add Project

#### <span style={{ color: '#167219' }}>_POST_</span>

Creates new project.

```url
https://translized.eu-4.evennode.com/project/add
```

| Params       | Type   | Required | Description           |
| ------------ | ------ | -------- | --------------------- |
| projectName  | String | True     | Name of project       |
| languageCode | String | True     | Primary language code |

<Tabs className="api-tabs">
<TabItem value="cURL" label="cURL">

```curl
curl -X POST -H "Content-Type: application/json" \
     -H "api-token: 42cf5fec-a74c-4a53-8ebc-86a4e52b7ce6" \
     -d "{"projectName":"Translized", "languageCode":"en"}" \
     https://translized.eu-4.evennode.com/project/add
```

</TabItem>
<TabItem value="js-fetch" label="Javascript Fetch">

```js
fetch('https://translized.eu-4.evennode.com/project/add', {

  headers: {
    'Content-Type': 'application/json',
    'api-token': '42cf5fec-a74c-4a53-8ebc-86a4e52b7ce6',

  body: JSON.stringify({

    languageCode: 'en',
```
